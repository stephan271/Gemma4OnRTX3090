# Contributing to Gemma4OnRTX3090

Thanks for your interest in improving this guide! 

## How to Contribute

1. **Bug Reports & Feature Requests**: Please open an Issue if you find a configuration that works better or a bug in the provided scripts.
2. **Pull Requests**:
   - For documentation improvements, feel free to open a PR directly.
   - For script changes, please ensure they have been tested on a Linux environment with an NVIDIA GPU.
3. **Benchmarks**: If you achieve different performance results on other hardware (e.g., RTX 4090), feel free to share them in the Issues or via a PR to the performance section.

## Code of Conduct

Please be respectful and helpful to others in the community.
